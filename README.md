# Carbon Upscale Style Guide

## Overview

This repository contains the official style guide for Carbon Upscale, providing comprehensive guidelines for our brand identity, visual elements, and communication standards. The style guide ensures consistency across all Carbon Upscale digital products, marketing materials, and communications.

## Purpose

The Carbon Upscale Style Guide serves as the single source of truth for our brand identity. It provides team members, partners, and contractors with clear guidelines on how to represent our brand. Consistent application of these standards helps us build a strong, recognizable brand that resonates with our mission of empowering individuals and organizations to address climate change through carbon footprint measurement, reduction, and offsetting.

## Contents

The style guide includes detailed specifications for:

- **Logo Usage**: Guidelines for proper implementation of our logo, including spacing, sizing, and variations
- **Color Palette**: Primary, secondary, and functional colors with hex codes and usage recommendations
- **Typography**: Font specifications for Fraunces, Inter, and Manrope with size and weight guidelines
- **Visual Elements**: Standards for data visualization, photography, illustrations, and iconography
- **Voice and Tone**: Communication principles and writing guidelines that reflect our brand personality
- **Applications**: Implementation examples across digital and print platforms

## How to Use This Guide

1. **Familiarize yourself with the contents**: Before creating any branded materials, review the relevant sections of the guide
2. **Reference the specifications**: Use the exact color codes, fonts, and measurements provided
3. **Follow the examples**: Use the application examples as templates for creating new materials
4. **Ask questions**: If you encounter situations not covered by the guide, please reach out to the design team
